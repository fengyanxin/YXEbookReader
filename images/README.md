# 图标资源说明

本目录需要放置以下图标文件：

## TabBar 图标（必需）

| 文件名 | 说明 | 尺寸建议 |
|--------|------|----------|
| bookshelf.png | 书架图标（未选中状态） | 81px × 81px |
| bookshelf-active.png | 书架图标（选中状态） | 81px × 81px |
| upload.png | 上传图标（未选中状态） | 81px × 81px |
| upload-active.png | 上传图标（选中状态） | 81px × 81px |
| settings.png | 设置图标（未选中状态） | 81px × 81px |
| settings-active.png | 设置图标（选中状态） | 81px × 81px |

## 其他图标（可选）

| 文件名 | 说明 |
|--------|------|
| default-cover.png | 默认书籍封面 |

## 图标设计建议

1. **风格**：简约线性图标或面性图标
2. **颜色**：
   - 未选中状态：灰色 (#95a5a6)
   - 选中状态：蓝色 (#3498db)
3. **格式**：PNG，透明背景
4. **尺寸**：81px × 81px（@3x），实际使用会自动缩放

## 临时解决方案

如果您暂时没有图标，可以：

1. 从 iconfont 等图标库下载
2. 使用在线图标生成工具
3. 暂时注释掉 `app.json` 中的 `tabBar` 配置

注释方法：
```json
// "tabBar": {
//   "color": "#95a5a6",
//   ...
// }
```
