# EPUB/PDF 解析功能测试指南

## 测试前准备

1. 确保项目已编译通过
2. 准备测试用的 EPUB 和 PDF 文件

## 测试文件要求

### EPUB 测试文件
- 未加密的标准 EPUB 文件
- 包含文本内容（非纯图片）
- 文件大小小于 50MB

推荐测试：
- 从 Project Gutenberg 下载的 EPUB
- 公版书 EPUB 文件

### PDF 测试文件
- 包含可提取文本的 PDF（非扫描件）
- 未加密的 PDF 文件
- 文件大小小于 50MB

推荐测试：
- 文本型 PDF（Word 导出）
- 网页保存的 PDF

## 测试步骤

### 1. 测试 EPUB 上传

1. 打开小程序
2. 点击底部"上传"标签
3. 点击上传区域
4. 选择"从聊天中选择"
5. 选择一个 EPUB 文件
6. 等待"正在处理..."提示消失
7. 查看是否显示解析结果：
   - ✅ 书名被正确提取
   - ✅ 作者被正确提取（如果有）
   - ✅ 内容预览显示文本
8. 点击"保存"

### 2. 测试 EPUB 阅读

1. 在书架中找到刚上传的 EPUB
2. 点击封面进入阅读
3. 检查：
   - ✅ 显示可阅读的文本内容
   - ✅ 可以调整字体大小
   - ✅ 可以切换主题
   - ✅ 阅读进度正常保存

### 3. 测试 PDF 上传

1. 点击底部"上传"标签
2. 点击上传区域
3. 选择一个 PDF 文件
4. 等待解析完成
5. 查看提示信息：
   - ✅ 成功：显示"PDF文本提取成功"
   - ⚠️ 失败：显示"PDF为图片型或加密"
6. 点击"保存"

### 4. 测试 PDF 阅读

1. 在书架中找到刚上传的 PDF
2. 点击封面进入阅读
3. 检查：
   - ✅ 成功：显示可阅读的文本
   - ⚠️ 失败：显示提示信息

## 预期结果

### EPUB
```
✅ 解析成功 → 显示书架卡片
✅ 进入阅读 → 显示文本内容
✅ 调整设置 → 样式正常变化
```

### PDF（文本型）
```
✅ 解析成功 → 显示书架卡片
✅ 进入阅读 → 显示提取的文本
```

### PDF（图片型）
```
⚠️ 解析部分成功 → 显示书架卡片
⚠️ 进入阅读 → 显示提示信息
```


### 常见错误信息

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| "不是有效的 EPUB 文件" | 文件损坏或格式错误 | 检查文件是否完整 |
| "EPUB解析失败" | 解压失败或文件结构异常 | 尝试其他 EPUB 文件 |
| "无法提取文本内容" | 图片型或扫描型 PDF | 使用 OCR 工具处理 |
| "文件过大" | 超过 50MB 限制 | 压缩文件大小 |
